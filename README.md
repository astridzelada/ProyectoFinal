El presente proyecto final fue desarrollado con el propósito de ofrecer una solución tecnológica integral para la gestión administrativa de un club de fútbol.
El sistema permite el acceso a tres tipos de usuarios: administradores, entrenadores y jugadores, cada uno con funcionalidades específicas según su rol dentro del club.

A través de esta plataforma, los administradores pueden gestionar información relacionada con eventos, pagos, equipos y usuarios; los entrenadores pueden consultar y administrar los equipos y eventos deportivos; mientras que los jugadores tienen acceso a la información de usuarios y datos relevantes de su participación en el club.

Este proyecto busca optimizar los procesos internos, facilitar la comunicación y organización, y promover una gestión más eficiente dentro del entorno deportivo.
